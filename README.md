📊 Courses Analysis Dashboard (Power BI)
📌 Overview

The Courses Analysis Dashboard is an interactive Power BI project designed to analyze course data and generate meaningful insights. It helps in understanding course performance, student engagement, and content effectiveness through visually rich and dynamic reports.

This dashboard enables users to explore trends, compare categories, and make data-driven decisions efficiently.

📸 Dashboard Preview

🎯 Objectives
Analyze course performance and sales
Track subscriber growth and engagement
Understand content duration and effectiveness
Identify top-performing categories and subjects
📊 Key Features
📈 KPI Cards (Sales, Subscribers, Courses, Content Duration)
🧾 Category-wise Sales Analysis (Paid vs Free)
📚 Subject-wise comparison (Business, Design, Music, Web Dev)
🔍 Interactive filters (Year, Subject, Level)
👨‍🎓 Subscriber and engagement insights
🏷️ Keyword analysis for course trends
📊 Clean and structured dashboard layout
🛠️ Tools & Technologies
Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization
